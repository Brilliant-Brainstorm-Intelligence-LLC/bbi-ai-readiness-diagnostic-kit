# License decision required before public release

Choose and add a real license before the first public release.

Candidates to review:

- **Apache-2.0:** permissive code reuse with an explicit patent grant.
- **MIT:** short and permissive.
- A dual-license approach if code and proprietary consulting templates need different treatment.

Do not publish with an accidental or unclear license. Repository visibility does not itself grant reuse rights, but ambiguity weakens adoption and contribution.
