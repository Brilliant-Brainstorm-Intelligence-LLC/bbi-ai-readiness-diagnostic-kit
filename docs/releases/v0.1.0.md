# BBI AI Readiness Diagnostic Kit v0.1.0

The first public release provides a transparent, evidence-first method for evaluating whether a specific AI use case is ready for implementation, requires further discovery, or should not proceed yet.

## Included

- Six readiness dimensions
- Explicit scoring weights
- Critical-dimension safeguards
- Evidence-count and missing-evidence warnings
- Deterministic JSON input
- Console output
- Markdown report generation
- Fully fictional sample assessment
- Unit-test coverage
- GitHub Actions verification across supported Python versions
- Public Proof Gate
- Contribution, security, support, and conduct standards

## Intended use

Use the kit to structure early AI-readiness conversations and identify missing business, data, oversight, governance, delivery, and evidence conditions.

## Boundaries

This release does not:

- certify compliance,
- approve an AI deployment,
- connect to private BBIOS data,
- replace legal, security, privacy, financial, accessibility, or domain review,
- use a model or external AI service,
- make automated implementation decisions,
- represent sample results as live operational telemetry.

## Verification

- Unit tests pass.
- The sample CLI assessment completes successfully.
- Markdown report generation completes successfully.
- GitHub Actions verifies supported Python versions.
