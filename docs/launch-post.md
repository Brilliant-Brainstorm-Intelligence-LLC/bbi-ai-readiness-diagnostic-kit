# Launch Post — BBI AI Readiness Diagnostic Kit v0.1.0

I've released the first public version of the **BBI AI Readiness Diagnostic Kit**.

Organizations often begin AI work by choosing a model or vendor before answering the more important questions:

- What business decision should improve?
- Is the necessary data available and trustworthy?
- Who retains human authority?
- What happens when the system is wrong?
- What evidence justifies moving forward?

The kit evaluates six practical dimensions:

1. Business value
2. Data readiness
3. Human oversight
4. Risk and governance
5. Delivery readiness
6. Evidence quality

It produces a transparent readiness score, dimension-level findings, evidence warnings, and a bounded recommendation.

The v0.1.0 release includes:

- a no-dependency Python scoring engine,
- fictional sample data,
- a command-line workflow,
- Markdown report generation,
- automated tests,
- GitHub Actions verification,
- and explicit governance limitations.

This is not a compliance certification or automated approval system. It is an evidence-first decision aid designed to improve the quality of the conversation before organizations build.
